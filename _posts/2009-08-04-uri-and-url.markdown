**URI - Universal Resource Identifier**  
统一资源标识符

**URL -Uniform Resource Location**  
统一资源定位符

**URN - Uniform Resource Name**  
统一资源名称

URI的通用格式为：

<code>协议:[//][[用户名[:密码]@]主机名[:端口号]][/资源路径]</code>

URL是URI的一个子集。URI包括URL和URN(Uniform Resource Name)两部分.
