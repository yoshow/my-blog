Windows 10 + Python 2.7

安装 Tornado  
<code>pip install tornado</code>

安装 MySQL Connector  
<code>pip install mysql-connector-python</code>

安装 MongoDB 客户端  
<code>pip install pymongo</code>

安装 SQLAlchemy  
<code>pip install sqlalchemy</code>
